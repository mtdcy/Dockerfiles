# nginx

## Quick Start

```shell
docker pull ghcr.io/mtdcy/nginx:latest

docker run -it -d --name nginx \
         -p 80:80 \
         -p 8080:8080 \
         ghcr.io/mtdcy/nginx:latest
```

Access nginx ui with http://<your_ip>:8080.
